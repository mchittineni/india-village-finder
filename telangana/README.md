# Telangana — Village Finder

Self-contained map app and dataset for **Telangana** (33 districts · 621 mandals ·
11,308 villages).

> **Live:** https://mchittineni.github.io/india-village-finder/telangana/web/

```
web/                 the map app — open web/index.html
  app.js, styles.css, index.html, config.js
  data/
    regions.json     districts + mandals (+ village counts) for the map
    villages.json    compact village list used for search
    meta.json        provenance, counts, live-LGD verification result
    districts.geojson, mandals.geojson   simplified boundary polygons
data/
  telangana_villages.csv   flat export — one row per village
```

This folder is generated by [`../scraper/`](../scraper/) — don't edit `web/` by hand;
change `scraper/web_template/` and re-run the pipeline. Data is the official
**Local Government Directory (LGD)**. See the [root README](../README.md) for the full
picture.
